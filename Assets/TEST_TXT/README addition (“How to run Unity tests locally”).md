## How to run Unity tests locally

1. Open **Window → General → Test Runner**
2. Select **Edit Mode** or **Play Mode**
3. Click **Run All** or **Run selected**
4. Save screenshots of the results to `TEST\_RESULTS/screenshots/`
   (Optional) Install **Code Coverage** package → Generate report → store under `TEST\_TXT/TEST\_RESULTS/`
5. If installed code coverage you can Access it **Window → Analysis → Code Coverage (**make sure that **burst** is not on, can be found in **Jobs)**
